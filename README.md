
# 🔦 Simulador da Lei de Malus 

## Introdução

O projeto apresenta um simulador da Lei de Malus, com dois polarizadores, uma fonte de luz e um sensor de leitura da intensidade da luz. 

A Lei de Malus indica que se uma luz não polarizada passa pelo primeiro polarizador, sua intensidade é dividida por 2. Essa representação ocorre no simulador quando o usuário diz que o ângulo do primeiro polarizador é ZERO

## Ferramentas e Instalação
### Ferramentas

O código foi feito inteiramente em PYTHON, utilizando pygame para desenhar as telas e adicionar as imagens referentes aos polarizadores, fonte de luz e sensor de luz.
As VERSÕES são:
> [!NOTE]
> - Python: 3.12.1
> - Pygame: 2.6.1
> - Pip: versão mais atualizada que tiver

### Instalação 

Para baixar o python, basta entrar no site <a href="https://www.python.org/downloads/"> Python Download </a> e clicar para fazer a instalação.

> Não se esqueça de clicar na caixa que diz "Add Python to PATH" para garantir seu funcionamento correto.

Para a instalação do VS Code, entre no site <a href="https://code.visualstudio.com/download"> VS Code Download </a> e clicar na instalação que seja referente à seu sistema operacional. (O original deste projeto é Windows).

E por fim, para instalar o pygame, basta digitar esse código dentro do Terminal do VS Code:

```

```

OU, caso necessário

```
pip3 install pygame
```
